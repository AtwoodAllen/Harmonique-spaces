# Harmonique Spaces 

A modern web project for **Decoration, Interior Designing, and Event Planning**, built with **HTML, CSS, and JavaScript**.  
This site demonstrates elegant design principles, responsive layouts, and interactive features — created as part of my coursework and portfolio.



##  Features
- **Homepage Hero Section**: Clean introduction with tagline and visuals
- **Services Grid**: Interior Design, Event Planning, and Decorations
- **Responsive Design**: Works seamlessly on desktop and mobile
- **Interactive Elements**: Smooth navigation, form validation, and image galleries
- **Future Dashboard**: Analytics and service overview for lecturer review



##  Technologies Used
- **HTML5** → semantic structure
- **CSS3** → styling, responsive layouts, animations
- **JavaScript (ES6)** → interactivity and dynamic features
